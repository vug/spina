Spina A Toy Molecular Dynamics Simulator and Visualizer

Introduction

This is a self-assigned sample project for my application to DESRES. My goal is to show my enthusiasm about the topics on which DESRES is working, and also to provide a public code sample and a demostration page for the interviewers to analyze, and maybe even enjoy. :-)

The project is named after Alessandro della Spina of Pisa who was an Italian Friar from early 14th century and mass produced eyeglasses first. :-)

Components

Components of Spina are following

  • Simulator: A toy simulator written in Python that runs MD simulations of N atoms which only interact with Lennard-Jones potential.
  • Visualizer. A browser based simulation results visualizer.
  • Notebooks: Interactive Jupyter Notebooks to explain how the system works.
